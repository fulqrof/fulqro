# FULQRO

FULQRO is an omnipresent structural intelligence system built with Next.js 14+, Tailwind CSS, MDX, TypeScript, and Recharts.

## Brand

### Colors
- Obsidian: `#141425`
- Fulcrum Amber: `#D4880A`
- Charcoal: `#2D2D2D`
- Forge White: `#E8E4DE`

### Typography
- Jost: headings and UI
- Source Serif 4: body
- JetBrains Mono: data

## Rules
- No bold in body text
- No em dashes
- No pure black
- All monetary values in USD
- LatAm depth in every report
